# CodeArena

A high-performance, gamified dashboard for LeetCode athletes. Track your mastery, compete in real-time, and visualize your journey from Easy to Hard with a sleek, minimalist interface.

## Features
- **Real-time Sync**: Get your latest LeetCode stats instantly.
- **Mastery Analysis**: Visualize your progress with creative donut charts.
- **Google Auth**: Secure and easy login.
- **Production Ready**: Optimized for hosting on Hugging Face Spaces.

## Development
```bash
# Backend
cd server && npm install && npm run dev

# Frontend
cd client && npm install && npm run dev
```
